# Processo seletivo - Desenvolvedor Full-Stack - Sênior - .NET (C#) e ReactJS

Bem vindo, candidat@.

Estamos felizes que você esteja participando do processo seletivo para a vaga de Desenvolvedor Full-Stack - Sênior - .NET (C#) e ReactJS - Soluções Digitais.

### A prova deverá utilizar as seguintes tecnologias:

- Linguagem de programação C# (.NET) para backend e React com Typescript para frontend.
- Banco de dados Microsoft SQL Server ou PostgreSQL.
- Testes unitários.
- GIT para versionamento da aplicação.
- Docker para a execução e isolamento das aplicações

### A aplicação deve contemplar os seguintes requisitos:

- O sistema deve seguir o Roteiro da Aplicação do documento encaminhado por e-mail.
- Você deverá modelar o layout da aplicação com os formatos, tipografias, cores e organização de layout que achar melhor.
- Você deverá se preocupar com a usabilidade da aplicação.
- Pense no contraste das cores para não atrapalhar na visualização.
- Implemente sempre que possível responsividade para adaptação em telas de computadores de diferentes tamanhos.
- As rotas no backend desenvolvidas devem contemplar testes unitários.
- Aplique os padrões de projetos que acha necessário para realização dos requisitos.

### Na etapa da entrevista deverá ser apresentado a aplicação em funcionamento.

- Instruções para a execução da prova será enviada por e-mail no horário definido em edital.
- Todo código desenvolvido deverá ser commitado neste repositório, não sendo aceito o envio da prova por outros meios.

### Será avaliado

- Facilidade no entendimento do código.
- Complexidade ciclomática do código.
- Divisão de responsabilidades das classes.
- Reutilização de código.
- Organização do projeto;
- Qualidade dos testes unitários implementados.
- Cobertura dos testes unitários.
- Desenvolvimento e funcionamento dos requisitos propostos.
- Utilização e configuração dos containers docker.
- Criatividade e inovação na solução proposta.
- Usabilidade do usuário.

### Informações extras

Subistitua esse arquivo readme.md no repositório do seu projeto no GitHub, para documentar a sua solução, bem como demonstrar as técnicas e linguagens utilizadas, além do escopo do projeto e como o usuário pode executar o seu sistema.
